## Usage

```bash
# Creates GNU makefiles with premake
$ ./vendor/premake5 gmake

# Compiles with GCC
$ make

# Runs demo
$ ./bin/Debug-linux-x86_64/Demo/Demo
```

### *Clean*

```bash
# Cleans all make compilation files
$ make clean

# Cleans all premake makefiles and folders
$ ./vendor/premake5 clean
```
